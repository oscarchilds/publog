# PubLog

Track your pub adventures with friends. Mark pubs you've visited, add notes about your experience, and explore where everyone's been.

## Features

- **Browse & Discover**: Explore pubs organized by area with map views
- **Track Visits**: Mark pubs as visited and add personal notes
- **See Progress**: Visual indicators showing how many pubs you've completed in each area
- **Compare Notes**: View other members' visits and their experiences
- **Mobile Friendly**: Responsive design that works great on phones

## Access

- **Anyone**: Browse pubs and view all visits (no login required)
- **Members**: Log in to track your own visits and progress
- **Admins**: Manage pubs, areas, and user accounts

## Getting Started

1. Ask an admin for a login account
2. Visit the site and log in
3. Explore areas and mark pubs as visited
4. See your progress and compare with friends
